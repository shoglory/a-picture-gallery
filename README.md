## 安装并使用
首先克隆到本地。然后执行
```
npm install
```
安装好依赖以后
```
npm run dev
```
打开本地http://localhost:8008 即可看到。
## 新的体会
由于前端技术发展日新月异,与老师所用的版本不同,所以在项目环境的配置上并没有使用脚手架工具,直接自己配置好webpack.config文件就直接上手开发了。同时,整个项目的语言我升级成了ES6,搭配babel使用,感觉非常爽！  
在开发过程中,顺着老师的整个流程,还是很容易的就完成了开发。
